# 🍸 Elixires de la Razón: Bar de Escuelas Económicas

> Experiencia digital interactiva y simulador cognitivo de coctelería intelectual y política. Basado conceptualmente en la investigación de **Ha-Joon Chang** (*Economics: The User's Guide*, 2015) y el ensayo epistemológico de *El Zorro y el Erizo* de **Isaiah Berlin** (1953).

Este proyecto es una **prueba de concepto premium** y autoportante que transforma las escuelas de pensamiento económico, filosófico y político en licores e ingredientes activos de coctelería para modelar arquitecturas cognitivas, tensiones distributivas y trade-offs complejos en tiempo real.

---

## 🎨 Características de Lujo Inmersivo

1. **El Aparador Alquímico (Hero)**: 12 botellas de boticario interactivas e hiperreactivas al cursor hechas con SVGs y CSS nítido. Su hover despliega tickets de catas refinadas que exponen el año, **ABV (Graduación Ideológica)**, notas organolépticas metafóricas de su doctrina y sus efectos secundarios sistémicos.
2. **La Carta de Cócteles Doctrinales**: Menú de 8 recetas premium recomendadas para resolver tensiones de la sociedad (ej. *Martillo Dialéctico*, *Libertad Líquida*, *Socialdemocracia Nórdica*, *Silicon Valley Sour*). Al hacer clic en un cóctel, un cajón lateral inmersivo se desliza, revelando un canvas nativo donde los fluidos de las escuelas se vierten y fusionan dinámicamente con física de partículas de salpicaduras y burbujas.
3. **Apothecary Cabinet of Problems**: Un mueble tridimensional de boticario antiguo. Al hacer clic en sus cajones de madera, estos se deslizan hacia el usuario para revelar pergaminos oficiales que comparan 3 remedios opuestos de escuelas alternativas para resolver la crisis de turno (*Gentrificación, IA, Inflación, Monopolios, Colapso Ecológico*), detallando su **pH Estatal** (acidez interventora) y su **toxicidad colateral** inevitable.
4. **Test de Alcoholemia Política**: Cuestionario de 12 reactivos de alta sofisticación académica divididos en tres fases (epistemología de Berlin, filtrado cruzado de afinidad ySliders continuos de trade-offs de alta tensión).
5. **Dossier de Resultados y Botella Dinámica**: 
   * Calcula tu **Arquetipo Cognitivo** contrastando tu autodeclaración con tu **Entropía real e Índice Herfindahl-Hirschman (HHI)** (*Zorro Sistémico, Erizo Doctrinal, Erizo Ecléctico, Zorro Camuflado*).
   * Genera dinámicamente tu **botella física virtual** con las capas de colores exactas de tu mezcla de afinidades.
   * Dibuja un **Gráfico de Radar nativo en Canvas** evaluando tu equilibrio entre Estado, Mercado, Equidad y Crecimiento.
   * Describe la **resaca ideológica y riesgos latentes** de tu pensamiento (ej. *Dogmatismo Agudo, Soledad del Ecléctico, Fricción Decisional*).
   * Permite **exportar e imprimir tu tarjeta oficial** en un formato vintage de lujo.

## 🚀 Despliegue en 1 Paso a GitHub Pages

Este proyecto ha sido programado en **Modo SIMPLE Premium (HTML5/CSS3/Vanilla JS autoportante en 4 archivos)**. Esto significa que:
* **Carga en 0.2 segundos** al no tener bundle-steps pesados de npm.
* **No contiene dependencias de compilación externas**, eliminando cualquier riesgo de error local en Windows o de CI/CD.
* **Funciona al instante** con solo abrir `index.html` en cualquier navegador del mundo.

### 📥 Instrucciones de Empuje Git (Stand-alone Repo)

Para subir esta carpeta e inicializarla como la rama principal (`main`) de tu repositorio exclusivo `economic-bar` y activar las GitHub Actions automáticas:

1. Abre tu terminal de PowerShell o Bash **directamente dentro del subdirectorio `main2/`**:
   ```bash
   cd C:\Dev\BAR-Economic\main2
   ```

2. Inicializa el repositorio Git local:
   ```bash
   git init
   ```

3. Agrega todos los archivos autoportantes:
   ```bash
   git add .
   ```

4. Haz el commit fundacional:
   ```bash
   git commit -m "feat: Initial commit of Economic Bar Speakeasy conceptual PoC"
   ```

5. Vincula tu repositorio remoto y empuja a producción en la rama `main`:
   ```bash
   git remote add origin https://github.com/willkwolf/economic-bar.git
   git branch -M main
   git push -u origin main
   ```

## ⚙️ Automatización en GitHub Actions

Hemos incluido un workflow preconfigurado en `.github/workflows/deploy.yml`. Al hacer el `git push`, GitHub Actions compilará la infraestructura estática y la desplegará de forma totalmente gratuita en:
```
https://willkwolf.github.io/economic-bar/
```

*(Asegúrate de ir a las configuraciones de tu repositorio `economic-bar` en GitHub: **Settings -> Pages -> Build and deployment -> Source** y seleccionar **GitHub Actions** para que el workflow tenga el permiso de escritura y despliegue).*
